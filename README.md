## How to Use

Using our Petrol Station Locator is straightforward:

1. **Homepage:** When you arrive at our homepage, you'll see the interactive map of Lagos.

2. **Map Interaction:** The map is marked with icons representing petrol stations across Lagos. You can zoom in and out to navigate the map effectively.

3. **Location Tracking:** If you allow location access, the map will center on your current location, making it even easier to find nearby petrol stations.

4. **Petrol Station Icons:** Each petrol station is marked with an icon that includes the station's name and address.

5. **Click for Details:** Click on an icon to view more information about the selected petrol station, including operating hours, available fuel types, and any additional services.

6. **Directions:** Once you've selected a petrol station, you can get directions to it from your current location using popular map navigation applications.

7. **Search Function:** Use the search bar to enter specific locations or areas in Lagos to find petrol stations in those regions.
